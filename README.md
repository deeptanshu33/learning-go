# Learning Go

This repository contains my notes, exercises, and experiments while learning Go.

The primary resource I'm following is **Learn Go with Tests** by Chris James:

* https://quii.gitbook.io/learn-go-with-tests/

In addition to the exercises from the book, this repository also includes:

* Small code snippets for experimenting with Go features
* Practice programs
* Solutions to exercises
* Other learning-related examples

The goal of this repository is to document my progress as I become more comfortable with Go and its ecosystem.
